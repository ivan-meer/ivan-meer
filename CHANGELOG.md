# Changelog

## [2.0.0] - 2026-07-18

### Features
- Полный редизайн GitHub Pages портфолио: новая дизайн-система «instrument panel» — светлый off-white фон, графитовые чернила, один киноварный акцент (OKLCH).
- Новая типографика с полной поддержкой кириллицы: Unbounded (дисплей), Onest (текст), PT Mono (данные/код) вместо Inter + JetBrains Mono.
- Авторская canvas-анимация «граф оркестрации агентов» в hero (вместо generic-частиц), с реакцией на курсор и статичным рендером при `prefers-reduced-motion`.
- Интерактивная схема слоёв full-stack AI (L1–L5) — аккордеон на главной.
- Страница услуг переписана: для каждой услуги — «когда нужно», «что входит», «результат»; добавлены форматы работы (аудит/прототип/под ключ/сопровождение) и FAQ.
- Line-mask reveal заголовка, варьируемые scroll-reveals, доступность: skip-link, aria-атрибуты, контент виден без JS (`no-js`-класс).
- README профиля переписан под новое позиционирование («системы, а не скрипты»), добавлена таблица слоёв стека.

### Design Rationale
- Прежний дизайн (тёмный неон, glassmorphism, частицы, эмодзи-иконки, нумерация секций 01/02) — узнаваемые паттерны AI-генерации; заменены на минимализм с воздухом и редакторской типографикой по методике skill impeccable (brand-регистр).
- Один акцентный цвет (эволюция прежнего оранжевого) вместо трёх неоновых — непрерывность бренда при смене эстетики.
- Путь пользователя: ценность за 5 секунд в hero → манифест «скрипты vs системы» → доказательство компетенции (схема слоёв) → проекты → процесс → CTA; на каждой странице один явный следующий шаг (Telegram).
